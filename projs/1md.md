# 1md

[GitHub repo](https://github.com/etar125/1md)

**1md** (emd, e1md) -- транслятор моего Markdown в одноимённый промежуточный код.
В рамках проекта также разрабатывается транслятор этого промежуточного кода в HTML -- **1md2ht**.

## Статус

Проект разрабатывается неспешно, в скором будущем будет архивирован.

## Планы

Собираюсь разработать свой язык разметки. До тех пор в 1md будут добавляться
всякие фичи из других Markdown.
